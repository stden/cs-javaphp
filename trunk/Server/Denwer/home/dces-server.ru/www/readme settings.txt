Place file dces-settings.php in this folder, and fill it with the following lines:

<?php
  $dces_mysql_host = "localhost:3306";
  $dces_mysql_user = "root";
  $dces_mysql_password = "";
  $dces_mysql_db = "dces";
  $dces_mysql_prefix = "dces_";
  $dces_logging = true;
?>

Don't forget to set appropriate values for the parameters
