Факториальная система. Вычитание чисел.
 
Требуется написать программу, которая осуществляет вычитание нескольких чисел в факториальной системе по заданным коэффициентам x1, x2, ... , xn всех чисел.
 В факториальной системе счисления основаниями является последовательность факториалов bk = k!, и каждое натуральное число a предствляется в виде:
a =x1 ·1!+x2 ·2! + … + xn ·n!
 
Входные данные:
Входной файл содержит несколько тестов.
Первая строка входного файла содержит одно натуральное число N – количество чисел.
Далее идут N строк – сами коэффициенты x1, x2, ... , xn всех чисел.
 
 
Выходные данные:
В каждой строке выходного файла находится перечисления коэффициентов  x1, x2, ... , xn  для разности чисел через пробел для очередного теста.
